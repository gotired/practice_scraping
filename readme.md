# Practice scraping

this repository is to practicing my scraping skills with `requests` and `beautifulsoup` with `python`

## Setup environment

on mac

1. create `venv`

```bash
python3 -m venv venv
```

2. activate `venv`

```bash
source venv/bin/activate
```

3. install requirements

```bash
pip install -r requirements.txt
```

4. start jupyter server

```bash
jupyter notebook
```
